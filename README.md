# Shopmate
ShopMate is a dynamic E-Commerce Web Application developed using Spring MVC Architecture. The project demonstrates complete shopping functionality including product browsing, cart management, user authentication, and invoice generation.

🚀 Features
👤 User Module

User Registration & Login

Session Management

Profile View & Update

Browse Products

Add to Cart

View Cart

Buy Now Option

Invoice / Receipt Generation

Print Bill

🛍 Product Module

Display All Products

Product Details Page

Dynamic Price Calculation

Image Display
